
# シューティングバトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
無双こうかとんをもとに作った宇宙が舞台のシューティングゲーム

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
* 敵キャラクターに関するクラス
* 爆発エフェクトのクラス
*ビームクラス
*　スコア表示のクラス

### 担当追加機能
* 敵の攻撃に関するatack関数(C0B22122)
* 当たると速度を阻害する敵モブ(c0b22038)
* 阻害された速度をスコアによって回復するアイテムの作成(c0b22038)
* 速度を表示する速度メーター(c0b22038)
* 残機表示クラス(C0A22160)
* ボスを出現させるクラス(C0A22056)
* itemを出現させる(C0A22036)
* itemでshieldの生成(C0A22036)
